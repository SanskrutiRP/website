OVERVIEW

Create a browser extension allowing users to vote for a domain categorizing it into various categories and get rewards. This enables them to pre-emptively protect their online activities from advanced threats in a collaborative way with a decentralized and tamperproof process using blockchain technology. Having a security & privacy focused framework for online activities & Pseudo-anonymous Reputation based systems for online interactions.

FEATURES

1. User - A person or a bot who can categorize a domain to be safe or unsafe.

Users must be registered on the Lazarus Network Platform to vote.

They must have a daily limit to submit their vote.

The limit should be increased/decreased based on their reputation.

The users can vote for these categories and tags:

Users can delete but can’t change their vote

Users can raise dispute against a vote submitted

User can appeal if they have been blocked from voting

Users can also appeal if they have been banned from voting

FEATURES

Moderator - A bot which checks whether a vote is genuine or fake. (Integrate ML Model to check webpage contents in real time)

Moderator checks for each vote by confirming these conditions

--Is the Domain reachable? If 404, A warning is given to the user

--Is the Domain registered? If not, then instant block the user from submitting any voting requests

Moderator logs each vote into the database based on the user’s wallet address and domain name

Moderators can block the user from voting again if the user receives 3 warnings and ban a user if both conditions of a vote aren't met.

An operator can unblock a user to start voting again after the appeal is addressed
